Heart Disease Prediction Model

This Heart Disease Prediction Model is built using a deep learning architecture for predicting the likelihood of having any cardiovascular diseases based on some clinical features. The model is trained on the UCI Cardiovascular Disease dataset, which includes features like age, cholesterol levels, and lifestyle factors such as smoking and alcohol consumption.

Key Metrics:
Accuracy: 73.37%
Precision: 74.88%
F1-score: 0.7261

Model Architecture: The model consists of several fully connected layers designed to capture non-linear relationships between the input features. It uses ReLU activation functions to introduce non-linearity and improve model learning.

How It Works: Input clinical data is provided through the web interface. The model processes this data and outputs a prediction (0 or 1) where 1 indicating the likelihood of heart disease and 0 indicating likelihood of not having a heart disease. The prediction helps identify individuals who may be at risk, providing valuable insights for timely medical intervention.
